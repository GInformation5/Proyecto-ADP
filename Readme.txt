Los únicos requisitos de software para que el usuario pueda usar la aplicación son:

1.1.	El directorio dist de la aplicación.

Este directorio contiene el archivo .jar de la aplicación y se encuentra en el repositorio de GitHub de la aplicación. Además, el archivo .jar de la aplicación puede ser ejecutado en muchos OS, por ejemplo, Windows, Android, Mac y GNU/Linux. Ademas el directorio dist contiene las librerías que son imprescindibles. Para ejecutar la aplicación únicamente hay que dar click al archivo .jar de la aplicación, es decir “Apliacion_Curso_Robotica.jar”.
 
 1.2. MySQL.

Es necesario que MySQL este instalado en la máquina del usuario.

1.3. El directorio con los archivos de la base de datos.
Es necesario que el directorio con los archivos de la base de datos este en el directorio Data de MySQL server. Este directorio se encuentra en el repositorio de GitHub.
